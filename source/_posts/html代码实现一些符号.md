## 整理常用的一些html特殊符号
在写html的时候经常会遇到一些特殊符号,例如`>`等,大家都知道直接写`<`或者`>`等代码会报错,原因是因为`<div></div>`,标签是用这个符号的,代码执行的时候会以为你只写了一半,所以报错,今天就整理一写这些特殊符号用html怎么表示


| html符号        | 特殊符号    |  含义  |
| --------   | -----:   | :----: |
|  `&lt;`        | &lt;      |   小于或显示标记      |
| `&gt;`         | &gt;      |   大于或者显示标记    |
| `&amp;`        | &amp;     |   用于显示其他特殊字符 |
| `&quot;`       | &quot;    |   引号              |
| `&reg;`        | &reg;     |   已注册             |
| `&copy;`       | &copy;    |   版权    |
| `&trade;`      | &trade;   |   商标    |
